# A股虚拟交易系统 - 待办功能 (TODO)

本项目未来的功能规划与优化方向：

## 1. 智能策略委托 (优先级：高)
### 三阶段阶梯止盈策略
- **功能描述**：在指定时间（如 14:50）根据当时市价自动拆解卖单。
- **核心逻辑**：
    - 若 `市价 < 止盈点1`：按 `止盈点1` 价格挂出 **1/3** 数量卖单。
    - 若 `止盈点1 <= 市价 < 止盈点2`：按 `止盈点2` 价格挂出 **2/3** 数量卖单。
    - 若 `市价 >= 止盈点2`：按 `止盈点3` 价格执行 **全仓** 清仓卖单。
- **技术实现**：
    - 后端：在 `trigger_matching` 中增加策略单解析逻辑，自动生成子委托。
    - 前端：增加“策略交易”面板，支持输入三个价位和触发时间。

## 2. 图表功能增强 (优先级：中)
- **K线图集成**：集成轻量级 K 线库（如 TradingView Lightweight Charts 或 ECharts），展示个股历史走势。
- **资产曲线**：在面板上展示每日总资产的变化趋势图。

## 3. 交易体验优化 (优先级：低)
- **快捷键支持**：支持快捷键快速切换股票、撤单等操作。
- **移动端适配**：优化 UI 在手机浏览器上的显示效果。
- **多账户支持**：支持在 `data.json` 中存储多个虚拟账户数据。

## 4. 系统管理
- **自动备份**：支持定时自动导出备份文件到指定目录。
- **历史回溯**：增加一个“复盘”模式，可以查看过去任意一天的持仓状态。
