# A股虚拟交易系统 (本地单机版)

这是一个基于 FastAPI 和 Vue 3 开发的极简 A 股虚拟交易系统，数据存储在本地 JSON 文件中。

## 核心功能
- **实时行情**：接入新浪财经公开接口。
- **虚拟交易**：支持买入、卖出委托。
- **自动撮合**：后台自动根据实时价格进行模拟成交。
- **T+1 规则**：模拟 A 股 T+1 结算规则。
- **JSON 存储**：所有持仓和资金记录保存在 `data.json` 中，交割单保存在 `history.csv` 中。

## 快速开始

### 1. 安装依赖
```bash
pip3 install fastapi uvicorn requests
```

### 2. 启动后端
```bash
python3 main.py
```

### 3. 访问前端
在浏览器中直接打开项目目录下的 `index.html` 即可。

## 数据文件
- `data.json`: 存储当前的账户资金、持仓信息和未成交委托。
- `history.csv`: 存储所有已成交的记录，可直接用 Excel 打开。

## 注意事项
- 默认初始资金为 100 万。
- 交易佣金设定为万分之三（最低5元），卖出时额外收取千分之零点五的印花税。
- T+1 结算需手动点击页面上的“T+1 日终结算”按钮（或由你根据需要调用 `/api/settle` 接口）。
